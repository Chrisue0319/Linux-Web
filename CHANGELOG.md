# 更新记录

## v2.0.0（2026-06-10）

### 架构重构

- 引入 SQLAlchemy ORM，审计日志持久化到 SQLite
- 采用清晰的分层架构：路由层 / 业务逻辑层 / 数据模型层 / 数据库访问层
- 新增 Pydantic v2 请求校验与自定义中文错误消息

### 新增功能

- 文件系统卷标查看与修改（ext/xfs/btrfs）
- 目录点击导航、面包屑、返回上级功能
- 创建目录后高亮显示新目录
- 关键系统目录保护（含子目录检测）

### 改进

- 422 验证错误统一翻译为中文
- 审计日志时间显示为北京时间（UTC+8）
- 错误弹窗区分验证错误和业务错误
- 前端过滤内部 JS 错误，避免向用户展示

### 测试

- 新增 FastAPI 集成测试（tests/test_api.py）
- 新增业务逻辑单元测试（tests/test_storage.py）
- 使用内存 SQLite 覆盖数据库依赖
